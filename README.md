# dy_fishing / 好好钓鱼
> 适配于 ESX 使用，钓鱼插件

[![pS7UKYR.png](https://s1.ax1x.com/2023/02/15/pS7UKYR.png)](https://imgse.com/i/pS7UKYR)

## 预定功能
* 自判断水域（其他区域无法钓鱼）
* 多个奖励模式（weapon，item，money）
* UI处理

## 使用指南

1. 前往合适水域
2. 使用鱼竿物品（'fishingrod'）


## 安装指南

### 自行下载
- 下载此资源并将其放入您的资源文件夹中。
  
### 增加到 cfg 文件
- 复制下行添加到您的 `server.cfg`:

```
ensure dy_fishing
```
